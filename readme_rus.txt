Face Sorter — сортировщик фотографий по лицам (GUI)

1. Запустите файл face_sorter_gui.py с помощью Python:
   > python face_sorter_gui.py

2. Выберите папку с фотографиями (jpg/png/tiff/heic и др.)

3. Выберите папку, куда сохранить отсортированные фото

4. Программа определит всех людей на снимках и разложит их
   по папкам Person_1, Person_2 и т.д.
